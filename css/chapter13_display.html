Inline elements

Inline Element는 콘텐츠의 길이만틈 공간을 차지한다.

여러 개의 Inline Element가 한 공간에 있을 경우, 옆으로 나열된다.

Inline Element는 width, height를 적용할 수 없다.

Inline-Block elements

Inline-Block element는 Inline Element와 동일하다.

단, width와 height를 조정할 수 있는 차이가 있다.
